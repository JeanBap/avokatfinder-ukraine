Kadastrovyy Nomer (Кадастровий номер)

The unique cadastral number assigned to each real estate parcel in Ukraine

Kadastrovyy nomer is the unique numerical identifier assigned to every parcel of land and registered real estate object in Ukraine. Managed by the State Geocadastre (Derzhavna Sluzhba Ukrayiny z Pytan Heodeziyi, Kartohrafiyi ta Kadastru), the cadastral number is the primary key for identifying any specific property in legal and administrative documents.

A cadastral number consists of ten digits structured as follows: region code (2 digits) + district code (2 digits) + community code (3 digits) + parcel number (4 digits). The number is assigned when a parcel is first registered in the State Land Cadastre and never changes, even if the property is sold, inherited, or renamed.

How It Works in Practice

All purchase contracts, mortgage agreements, and other legal documents relating to a property must reference its cadastral number. Without a valid cadastral number, the property cannot be registered in the State Register of Real Property Rights.

Buyers should verify the cadastral number by checking both the State Land Cadastre (publichnaya-kadastrovaya-karta.gov.ua) and the State Register of Real Property Rights. The two systems should have consistent data for the same property — discrepancies indicate potential problems.

BTI LegacyProperties without cadastral numbers must be registered first

Some properties — particularly old apartments or buildings — may lack a cadastral number if they were registered only in the old BTI system. In such cases, the seller must first register the property in the State Land Cadastre to obtain a cadastral number before the purchase contract can be executed.

Common Mistake: Buyers sometimes proceed to a notary appointment without first verifying that the property has a valid cadastral number. If the number is missing or incorrect, the notary cannot complete the transaction and the appointment must be rescheduled.
Expert Tip: Use the public cadastral map (publichnaya-kadastrovaya-karta.gov.ua) to verify the property's boundaries and confirm that the cadastral number in the seller's documents matches the parcel you are buying. Discrepancies are easily spotted on the map.

Frequently Asked Questions

What is a cadastral number in Ukraine?

It is a unique 10-digit identifier assigned to every land parcel and registered real estate object, used to identify property in all legal documents and registration systems.

Can a cadastral number change after a property is sold?

No. The cadastral number is permanent and does not change when a property is sold, inherited, or otherwise transferred.

What if a property doesn't have a cadastral number?

The seller must register the property in the State Land Cadastre to obtain a number before the purchase contract can be executed. This process takes time and should be completed during the due diligence phase.
